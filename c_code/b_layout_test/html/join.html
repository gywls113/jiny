쿠팡

회원가입에 필요한 개인정보를 입력하시오.

  • 이메일을 입력하세요.

  • 영문/숫자/특수문자 2가지 이상 조합 (8~20자)

    3개 이상 연속되거나 동일한 문자/숫자 제외

    아이디(이메일) 제외

  • 확인을 위해 새 비밀번호를 다시 입력해주세요.

  • 이름을 정확히 입력하세요.

  • 휴대폰 번호를 정확하게 입력하세요.

회원가입하기

본인은 만 14세 이상이며, 쿠팡 이용약관, 전자금융거래이용약관, 개인정보 수집 및 이용, 개인정보 제공 내용을 확인 하였으며, 동의합니다.
